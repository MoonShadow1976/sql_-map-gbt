# 模型使用ollama
#### 模型安装
```
ollama run sqlcoder:latest
```
# 使用conda命令安装python环境
```
conda create -n MapGBT python=3.10
```
# 对应需要安装的pip，详见requirements.txt
```
pip install ollama
pip install gradio
pip install psycopg2
pip install geopandas
pip install matplotlib
```
# 运行程序请运行web.py
```
your_python.exe web.py
```
